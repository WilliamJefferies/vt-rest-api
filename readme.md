# VT Rest API
Simple rest API for demonstration of node skills using a simple strategy design pattern and no db wrapper.

### Getting Started
##### Install & Start
```npm install && npm start```
#### Debug
```npm run debug```

### Calling the API
##### Get lotcodes
```/api/lotcodes```
##### Get breakdown strategies
```/api/breakdown-strategies```
##### Get breakdown
```/api/breakdown/${breakdown-strategy}/${lotcode}```
